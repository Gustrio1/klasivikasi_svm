BAB 1
Pengantar Sistem

Tentang Sistem Ini

Sistem Klasifikasi Hafalan Al-Qur'an adalah aplikasi web berbasis Laravel 11 yang dilengkapi dengan layanan prediksi berbasis Python (SVM Linear). Sistem ini dirancang untuk membantu lembaga pendidikan Al-Qur'an dalam:

  • ✅ Mencatat dan memantau progress hafalan siswa secara terstruktur per surah dan per semester
  • ✅ Melakukan klasifikasi kelulusan otomatis (Lulus / Tidak Lulus) menggunakan algoritma SVM Linear
  • ✅ Mengelola data guru, siswa, dan media belajar dalam satu platform terintegrasi
  • ✅ Menghasilkan laporan PDF hasil evaluasi hafalan per siswa
  • ✅ Memantau kinerja model SVM melalui log evaluasi (akurasi, precision, recall, F1-score)

BAB 2
Instalasi dan Konfigurasi

Kebutuhan Sistem (Requirements)

Software Wajib

  • 🔧 PHP ≥ 8.2
  • 🔧 Composer ≥ 2.x
  • 🔧 Node.js ≥ 18 & NPM
  • 🔧 Python ≥ 3.8
  • 🔧 MySQL ≥ 5.7 / MariaDB ≥ 10

PHP Extensions

  • ✓ php-pdo / php-mysql
  • ✓ php-json
  • ✓ php-mbstring
  • ✓ php-openssl
  • ✓ php-gd (untuk PDF)

Langkah Instalasi

1

Ekstrak / Clone Proyek

cd D:\riska
git clone <repo-url> web-svm
cd web-svm
2

Install Dependensi PHP

composer install
3

Install Dependensi Node.js

npm install
4

Konfigurasi Environment

copy .env.example .env
php artisan key:generate

Edit file .env dan sesuaikan:

DB_DATABASE=svm_hafalan
DB_USERNAME=root
DB_PASSWORD=

PYTHON_PATH=python
5

Migrasi dan Seed Database

php artisan migrate --seed
ℹ️ Seeder akan membuat akun admin default.
6

Setup Python SVM

cd svm_service
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
7

Jalankan Server

# Terminal 1
php artisan serve

# Terminal 2
npm run dev
✅ Akses sistem di: http://127.0.0.1:8000

BAB 8
Teori dan Algoritma SVM

8.1 Fitur Input (Variabel X)

FiturSimbolSumber DataKeterangan
Total Surah UnikX₁DataHafalanJumlah surah berbeda dalam satu semester (pengaruh dominan)
Usia SiswaX₂Profil SiswaUsia dari tahun lahir (pengaruh sangat kecil)
ID MediaX₃Modus MediaMedia paling sering digunakan (modus)

8.2 Formula Klasifikasi SVM Linear

FUNGSI KEPUTUSAN (DECISION FUNCTION)
f(x) = (W₁ × X₁) + (W₂ × X₂) + (W₃ × X₃) + b
W₁ = 1.0 (bobot utama — total surah)
W₂ = -0.01 (bobot kecil — usia)
W₃ = 0.01 (bobot kecil — media)
b = -29.5 (bias/threshold)

Jika f(x) ≥ 0 → LULUS
Jika f(x) < 0 → TIDAK LULUS
CONTOH PERHITUNGAN
Siswa: 32 surah, usia 12, media ID 2
f(x) = (1.0 × 32) + (-0.01 × 12) + (0.01 × 2) + (-29.5)
f(x) = 32 − 0.12 + 0.02 − 29.5 = 2.4

f(x) ≥ 0 → LULUS
Confidence = min(0.5 + 2.4/10, 0.99) = 0.74

8.3 Parameter Training SVM (Subgradient Descent)

ParameterNilaiFungsi
Learning Rate (η)0.01Ukuran langkah update bobot
Lambda (λ)0.001Regularisasi L2 mencegah overfitting
Max Epoch1000Batas iterasi training
Tolerance1×10⁻⁶Ambang early stopping
FUNGSI LOSS (HINGE LOSS + L2)
L = (λ/2)||w||² + (1/n) Σ max(0, 1 − yᵢ·f(xᵢ))

BAB 9
Alur Proses Klasifikasi
1

Input Hafalan oleh Guru

Guru mencatat hafalan siswa (nama surah, jumlah ayat, media, periode semester).

2

Agregasi Data per Semester

Sistem mengumpulkan semua hafalan siswa dan menghitung: total surah unik, modus media, usia siswa.

3

Eksekusi Python SVM

python svm_service/script_prediksi.py <total_surah> <usia> <id_media>
4

Prediksi dan Confidence

Python menghitung f(x), menentukan kelas, dan confidence score. Output dikembalikan dalam JSON.

5

Simpan Hasil Klasifikasi

Hasil (kelas, confidence, vector SVM, model aktif) disimpan ke tabel hasil_klasifikasi.

6

Tampilkan ke Guru & Siswa

Hasil dapat dilihat oleh guru dan siswa melalui menu Hasil Klasifikasi.

BAB 12
Frequently Asked Questions (FAQ)
?

Mengapa siswa dengan 30 surah tidak selalu Lulus?

Formula: f(x) = (1.0 × 30) + (-0.01 × usia) + (0.01 × media) + (-29.5). Jika usia tinggi, bisa menghasilkan f(x) negatif meski kecil pengaruhnya. Pada umumnya 30 surah sudah cukup untuk Lulus.

?

Apakah surah sama bisa disetorkan lebih dari sekali?

Ya. Namun untuk klasifikasi, sistem hanya menghitung jumlah surah unik, bukan total setoran.

?

Siapa yang bisa membuat laporan PDF?

Hanya Admin dan Guru. Siswa hanya dapat melihat dan mengunduh laporan yang sudah dibuat.

?

Bagaimana cara mengubah password siswa?

Guru (atau admin) dapat edit data siswa dan mengisi field "Password Baru" + "Konfirmasi Password".

?

Apa perbedaan "Perhitungan SVM" dan "Hasil Klasifikasi"?

Perhitungan SVM: halaman detail proses training (bobot, loss, confusion matrix). Hasil Klasifikasi: prediksi aktual untuk setiap siswa berdasarkan data hafalan nyata.

Klasifikasi SVM gagalPython tidak terinstall / path salahPastikan PYTHON_PATH=python di .env. Test: python --version
